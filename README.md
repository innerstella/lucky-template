# ⚙️ 주요 기능
1. 문항별 글자수 계산
2. 문항별 글자수 제한 설정
3. 메모 기록

<br/><br/>

# 📝 사용 방법
- 작성한 내용은 사용자의 브라우저에만 저장되며, 별도의 DB에 보관되지 않습니다.
1. 기업명 입력
2. 문항 입력
3. 글자수 설정
4. 메모 및 답변 작성
